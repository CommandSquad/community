# COMMAND SQUAD
## Chrome Remote Desktop Setup Guide
### Remote Into Your PC to Help Seed from Anywhere

> Whether you're at work, traveling, or away from home — this guide will help you securely access your gaming PC remotely so you can launch Squad and help keep the server seeded.

---

# 🎯 What This Lets You Do

With Chrome Remote Desktop, you can:

- Access your home PC remotely
- Launch Squad from anywhere
- Join the server to help seed
- Restart the game if needed
- Check Discord or server status
- Keep contributing even while away from home

This is especially useful for CMD Members helping maintain daytime seed momentum.

---

# ✅ Requirements

Before starting, make sure you have:

- A Windows gaming PC left powered on at home
- Google Chrome installed
- A Google account
- Stable internet at home
- Permission to install software on the PC

---

# 🔒 Security Recommendations

Before setting this up:

- Use a **strong Google password**
- Enable **2-Factor Authentication (2FA)** on your Google account
- Never share your remote PIN with anyone
- Only remote into PCs you personally own/control

Chrome Remote Desktop is generally very safe when configured properly.

---

# 🖥️ STEP 1 — Install Google Chrome

If Chrome is not already installed:

## Download Chrome
https://www.google.com/chrome/

Install Chrome and sign into your Google account.

---

# 🌐 STEP 2 — Open Chrome Remote Desktop

Visit:

https://remotedesktop.google.com/access

---

# ⚙️ STEP 3 — Enable Remote Access

1. Click **"Turn On"** under **Set up remote access**
2. Download the Chrome Remote Desktop Host installer
3. Run the installer
4. Allow permissions when prompted

Once installed:

- Give your PC a recognizable name

### Examples
- CMD-Gaming-PC
- MainRig
- Squad-Seeding-PC

---

# 🔐 STEP 4 — Create a Secure PIN

You will be prompted to create a PIN.

## Recommendations
- Minimum 6 digits
- Preferably 8+ digits
- Avoid birthdays/simple numbers

### Examples
- ❌ 123456
- ❌ 111111
- ✅ Random unique PIN

This PIN will be required every time you connect remotely.

---

# 📱 STEP 5 — Install on Your Phone or Laptop

## Mobile Apps

### iPhone / iPad
https://apps.apple.com/us/app/chrome-remote-desktop/id944025852

### Android
https://play.google.com/store/apps/details?id=com.google.chromeremotedesktop

You can also simply use another PC with Chrome installed.

> Sign into the SAME Google account used on your gaming PC.

---

# 🚀 STEP 6 — Connect to Your Home PC

1. Open Chrome Remote Desktop on your phone/laptop
2. Your gaming PC should now appear in the list
3. Click/tap the PC
4. Enter your PIN
5. Connect

Your desktop should now appear remotely.

---

# 🎮 STEP 7 — Launch Squad & Help Seed

Once connected:

1. Open Steam
2. Launch Squad
3. Join the CMD server
4. Spawn in / AFK seed as needed

---

# ⚡ Recommended "Always Ready" Setup

For the smoothest experience:

## Prevent Your PC From Sleeping

### Windows
1. Open **Settings**
2. Go to **System**
3. Select **Power & Sleep**
4. Set:
   - Sleep = Never
   - Screen Off = Optional

---

# 🌐 Optional Performance Improvements

For a better remote experience:

- Use wired Ethernet on the gaming PC
- Maintain decent upload speed (10 Mbps+ preferred)
- Keep Windows updated
- Reboot occasionally

---

# 🛠️ Common Issues

## PC Shows Offline

Usually caused by:
- Computer sleeping
- Windows restarting after updates
- Internet outage
- Chrome signed out

---

## Black Screen

Try:
- Disconnecting/reconnecting
- Restarting Chrome
- Updating GPU drivers later

---

## Mouse/Keyboard Not Responding

Usually fixed by:
- Clicking inside the remote window
- Reconnecting the session
