# 📡 About Command Squad

Command Squad is a Squad community focused on:

- Coordinated gameplay
- Teamwork and communication
- Strong leadership
- Fair and consistent administration
- Long-term community building
- New player friendly experiences

Our goal is simple:

> Create a high-quality Squad experience that players genuinely want to return to.

We are not a hardcore milsim community, nor are we a chaos server with no standards.

Command Squad aims to strike the balance between:
- structured gameplay
- approachable community culture
- competitive teamwork
- sustainable administration

---

# 🎯 Purpose of This Repository

This repository serves as the public knowledge base for the Command Squad community.

Here you'll find:
- setup guides
- onboarding resources
- server information
- community documentation
- branding/media assets
- public-facing resources

This repository intentionally excludes sensitive operational or administrative systems.
